## Github Finder

### Github

* Create a [Github OAuth Application](https://github.com/settings/developers)*
* In **Home Page URL**
    * http://localhost:3000
* In **Authorization Callback URL**
    * http://localhost:3000

* Take note of:
    * `Client ID`
    * `Client secrets`

* Rename `.env.example` to `.env`
    * Enter your `Client ID` and `Client Secrets`

Install dependencies
* `npm install`

Run the project
* `npm start`
